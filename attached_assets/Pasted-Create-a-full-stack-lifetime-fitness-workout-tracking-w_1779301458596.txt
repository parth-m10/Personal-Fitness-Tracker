Create a full-stack lifetime fitness workout tracking web application.

Tech stack:
- Frontend: React + Tailwind CSS + Framer Motion
- Backend: Node.js + Express
- Database: SQLite or PostgreSQL
- Charts: Recharts with animated transitions
- Icons: lucide-react
- UI should be clean, premium, modern, visually appealing, responsive, animated, and dashboard-style with a high-quality fitness app feel.

Important:
Before coding, first create a clear file structure and implementation plan. Then implement step by step. Ensure the app runs without errors.

Main concept:
This is NOT a 5-day-only app.
The app should use a 5-day workout plan as a repeating lifetime workout cycle.

The user should be able to use this app for months and years.
After Day 5 is completed, the app should automatically move back to Day 1 as the next workout cycle.
Each completed 5-day round should be counted as one workout cycle.

Core logic:
- The workout plan has Day 1 to Day 5.
- These 5 days repeat forever.
- The app should track:
  - Cycle number
  - Workout day number
  - Actual calendar date
  - Completion status
  - Exercise-wise progress over time
- If the user misses a day, the app should NOT reset.
- The app should continue from the next pending workout.
- Example:
  - If Day 3 was last completed, the next workout should be Day 4.
  - If Day 5 was last completed, the next workout should be Day 1 of the next cycle.

Workout cycle rules:
- Day 1 → Day 2 → Day 3 → Day 4 → Day 5 → Day 1 again
- Every time Day 5 is completed, increment cycle number by 1
- Missed days should not break the cycle
- User should also be able to manually skip a workout day
- If skipped, save it as skipped and move to the next workout day

Default workout plan:

Day 1: Chest + Shoulders + Core
- Warm-up: jumping jacks + steam engine + light cycling, 10 min
- 45-degree inclined pushups, 4 sets, 8–12 reps
- Negative pushups, 3 sets, 5–6 reps
- Knee pushups, 3 sets, 8–10 reps

- Crunches, 3 sets, 15 reps
- Plank, 3 sets, 30–45 sec
- Cycle, 25–30 min
- Cooldown, 8–10 min

Day 2: Legs + Obliques
- Warm-up: cycle + jumping jacks, 10 min
- Squats, 4 sets, 12–15 reps
- Lying leg abductions, 3 sets, 15 reps per side
- Lying leg scissors / hip abduction-adduction, 3 sets, 15–20 reps
- Toe raises for calves, 4 sets, 15–20 reps
- Bridges, 3 sets, 15 reps
- Side planks, 3 sets, 20–35 sec per side
- Mountain climbers, 4 sets, 30 sec
- Cycle, 25–30 min
- Cooldown, 8–10 min

Day 3: Arms + Core
- Warm-up: cycle + arm circles, 10 min
- Bicep curl, 3 sets, 10–12 reps, default weight 5 kg
- Hammer curl, 3 sets, 10–12 reps, default weight 5 kg
- Cross-body hammer curl, 3 sets, 10 reps per side, default weight 5 kg
- Tricep kickback, 3 sets, 12–15 reps, default weight 3 kg

- Bicycle crunches, 3 sets, 20 total reps
- Flutter kicks, 3 sets, 30 sec

- Cycle, 30–35 min
- Cooldown, 8–10 min

Day 4: Chest + Legs + Conditioning
- Warm-up: jumping jacks + cycle, 10 min
- Full pushup attempt, 4 sets, 3–5 partial reps
- Negative pushups, 3 sets, 5 reps
- Knee pushups, 3 sets, 10–12 reps
- Squats, 4 sets, 12 reps
- Bridges, 3 sets, 15 reps
- Wood chopper without dumbbells, 3 sets, 15 reps per side
- Steam engine, 3 sets, 30–40 sec
- Cycle, 25–30 min
- Cooldown, 8–10 min

Day 5: Full Body + Core + Longer Cardio
- Warm-up: cycle + jumping jacks, 10 min
- Standing shoulder press, 3 sets, 10–12 reps, default weight 5 kg
- Lateral raises, 3 sets, 12–15 reps, default weight 3 kg

- Superman, 3 sets, 12–15 reps

Exercise tracking:
- Track every strength exercise by:
  - Sets
  - Reps
  - Weight lifted in kg
  - Total volume = sets × reps × weight
- For bodyweight exercises, allow weight as 0 or bodyweight.
- Warm-up and cooldown exercises should not require weight tracking.
- Cycling should track:
  - Duration
  - Intensity: Easy / Moderate / Intervals / Hard
  - Approx calories burned
- Core timed exercises like plank and side plank should track duration.
- Bodyweight exercises should track reps, sets, and optional bodyweight.
